# Office-files

Central hub for office work: projects, client site builds, templates, assets, and training materials.

Overview:
- Projects & client sites: finished and in-progress site folders ready for deployment or demo.
- Assets: fonts, images, styles, and scripts used across projects.
- Templates & examples: reusable layouts and components.
- Training & notes: sample work, exercises, and documentation for skills development.

Intent: Organize and publish these folders to GitHub as an archive and portfolio.

> **Note:** This is a dynamic workspace. New demo folders are added regularly as I learn and practice. An organized archive for finished and in-progress work.
